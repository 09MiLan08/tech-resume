独自上手全栈开发领域，熟练运用 Vue3 构建多端响应式前端界面，结合 FastAPI/Django 实现高效后端服务，能独立完成从需求分析、架构设计到部署上线的全流程开发。注重代码规范与性能优化，具备解决实际业务问题的能力。

## 技术技能

- 核心框架：Vue3（Composition API、Pinia、Vue Router）、Vite、VitePress
- 开发能力：HTML5/CSS3（Flex/Grid）、JavaScript（ES6+）、ECharts 可视化
- 性能优化：v-memo 缓存、懒加载、多端自适应适配

- 核心能力：RESTful API 设计、权限控制、数据接口开发

### 数据库

- 基础：MySQL 增删改查、索引优化
- 工具：Navicat、Django ORM

### 工具与部署

- 版本控制：Git（分支管理、提交规范）
- 部署：GitHub/Gitee Pages、Vercel
- 辅助工具：Cursor、VS Code 插件生态
